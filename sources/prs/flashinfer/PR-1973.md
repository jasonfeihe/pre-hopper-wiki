# Feature: Add support for L40 FusedMoE in cutlass path

Adds an Ada Lovelace L40 (sm89) FusedMoE path in the CUTLASS backend. Because the L40 lacks Hopper's TMA/wgmma path, it dispatches to an sm_89 CUTLASS GEMM configuration using FP8 (E4M3) tensor-core MMA tuned for Ada's 4th-generation Tensor Cores (compute capability 8.9).
