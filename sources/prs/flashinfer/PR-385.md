# bugfix: Fix invalid kernel configuration for sm86

Fixes an invalid launch configuration on Ampere A10 (sm86): the decode attention kernel requested more shared memory than sm_86's ~100 KB/SM opt-in limit (smaller than sm80's 164 KB). The fix guards smem sizing on __CUDA_ARCH__ == 860 and clamps pipeline stages for A10's budget.
