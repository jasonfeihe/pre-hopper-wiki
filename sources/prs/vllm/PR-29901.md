# [Kernel][Quantization][MoE] add marlin kernel support for turing (sm75)

Adds a Marlin INT4 weight-only quantized MoE/GEMM kernel path for Turing (sm75, e.g. T4). The Turing branch is guarded on __CUDA_ARCH__ == 750 and uses sub-byte INT4 mma.sync with ldmatrix operand loads to drive Turing's second-generation Tensor Cores, extending Marlin (previously sm80+) down to the T4.
